# Библиотека проектных иконок Alterios

Файлы экспортированы из прямой папки `public` и сохранены без исходных project-local UUID.
Перед применением в проекте MCP должен проверить состав иконок проекта и загрузить только отсутствующие файлы через `alterios_ensure_project_icon_library`.
Все SVG в библиотеке подготовлены по стандарту Google Fonts Icons: UI Size `16`, цвет `#4B77D1`.
Фактические SVG `width`/`height` сохраняются такими, как их отдает Google Fonts при скачивании; вручную размер не переписываем.

Полный стандарт выбора иконок и действий: `docs/alterios-icons-and-actions-catalog.md`.
