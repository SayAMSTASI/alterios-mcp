# Source Map

Use these repo documents as evidence sources:

- `../../../docs/form-surface-inventory.md`: live form surface totals, cell types, action types, and form-kind counts.
- `../../../docs/form-surface-inventory.json`: machine-readable form matrix.
- `../../../docs/form-surface-ux-and-icons.md`: UX guardrails for form layout, action order, role/style/source checks.
- `../../../docs/view-format-inventory.json`: machine-readable view format matrix for table/grid/list/reference/gantt/leaflet/calendar settings.
- `../../../docs/view-format-ui-evidence-2026-07-11.md`: UI edit/preview evidence for all confirmed view formats.
- `../../../docs/material-types-and-view-types-research.md`: live findings for experimental view `settings.title`, `ref source=view`, joined table views, populated mnames, and person/ref display behavior.
- `../../../docs/alterios-entity-surface-catalog.md`: entity-level surface rules for content types, fields, views, forms, actions, scripts, reports, and relation scenarios.
- `../../../docs/script-bpmn-linkage.md`: task form, manual-script action, process action, and BPMN formKey cross-checks.
- `../../../docs/script-bpmn-linkage.json`: machine-readable script/BPMN refs for task/form cross-checks.
- `../../../docs/report-tab-openid-ui-research-2026-07-10.md`: `openId` and `dataId: [openId]` report/form behavior.
- `../../../docs/alterios-icon-standards.md`: icon standard used when action placement touches icons.

Use `alterios-form-surface-check` for local form JSON validation where available.
